## 🎯 Problem Statement
Analyze demographic and socio-economic factors to determine what influences whether an individual earns more than 50K per year.

---

## 📂 Dataset Features
- Age
- Education
- Occupation
- Workclass
- Hours per week
- Income

---

## 🛠️ Tools & Libraries
- Python
- Pandas, NumPy
- Matplotlib, Seaborn

---

## 🔍 Steps Performed
✔ Data Cleaning  
✔ Handling Missing Values  
✔ Exploratory Data Analysis (EDA)  
✔ Data Visualization  

---

## 📊 Key Insights
- Individuals with higher education levels are more likely to earn >50K  
- Working more hours per week shows a positive correlation with income  
- Certain occupations have significantly higher income distribution  
- Age plays an important role in earning potential
